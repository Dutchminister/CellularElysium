# Conways Game of Life ScreenSaver

## Description
Give life to your screensaver!

## Installation
1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage
1. Run the program:
    ```bash
    python3 ./conways_game_of_life.py
    ```

## Build
1. Build executable:
    ```bash
    pyinstaller --onefile conways_game_of_life.py
    ```
